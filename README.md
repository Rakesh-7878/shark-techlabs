# Shark-Tech Labs

A modern, interactive website for Shark-Tech Labs featuring an animated ocean particle background and responsive design.

## Features

- **Interactive Ocean Background**: Animated particle system with connecting lines
- **Responsive Design**: Mobile-first approach with modern UI/UX
- **Dark Theme**: Sleek black and white color scheme
- **Smooth Animations**: CSS transitions and JavaScript animations
- **Professional Layout**: Clean, modern design with sections for Mission, Products, and Team

## Technologies Used

- HTML5
- CSS3 (Custom styling with modern features)
- JavaScript (Canvas API for particle animations)
- Google Fonts (Inter and Space Grotesk)

## Getting Started

1. Clone the repository
2. Open `index.html` in your web browser
3. Enjoy the interactive ocean background!
